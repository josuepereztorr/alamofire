HTTP Request Methods with Alamofire 5

What is Hypertext Transfer Protocol (HTTP)?

According to the Mozilla Web Docs, HTTP is an application communication protocol that follows the client-server model that is used to fetch resources of a network. The HTTP protocol has a set of request methods that specify an action to do on a resource. Some methods include GET, POST, PUT, and DELETE.

Source: https://developer.mozilla.org/en-US/docs/Web/HTTP

What is Alamofire?

Alamofire is a Swift networking library that is based on Apple's Swift Foundation Framework (application base layer framework). Alamofire simplifies common network tasks and makes working with the HTTP communication protocol easier.
