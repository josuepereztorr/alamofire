Getting Started

Start by creating a new Xcode project either by clicking on Create a new Xcode project from the Xcode welcome window (on startup) or by going to File -> New -> Project.

Next, a window will appear asking you to Choose a template for your new project: select App and click Next.

Select the options for the project. For the purpose of this tutorial I chose the following options for my project. After that click Next.

Select the location you wish to store your project and click Create.
Note: For this project you can uncheck Create a Git repository on my Mac

At this point the initial project setup is complete. The starting template comes with an initial ContentView.swift SwiftUI file that contains an Image() and a Text() view. In addition, we have the vindecoderApp.swift which is the entry point in our application.
