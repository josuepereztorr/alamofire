Testing the App

Run the App Simulator

On the file explorer (left side of Xcode), click on the root directory Product, then select Run this should build and run your project. After this an iphone simulator should open up to try your application. There should be no VIN on the TextField view and the results should show our mock vehicle object.

To test the API, input a VIN and click Decode. This should now show all the information from the VIN on the bottom view.
