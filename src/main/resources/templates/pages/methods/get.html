GET Method using Alamofire 5

What is an HTTP GET method?

An HTTP GET method is used to request data (a resource) from an external server but, it should not change tha state of the server.

Goal

The goal of this tutorial will be to create a VIN (Vehicle Identification Number) decoder application using the Alamofire HTTP networking library and a external API.
